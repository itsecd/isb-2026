## Задание 1: Шифрование текста

### Файлы для проверки:
| Файл | Описание |
|------|----------|
| `task_1/main.py` | Исходный код программы |
| `task_1/key.txt` | Ключ шифрования |
| `task_1/text.txt` | Исходный текст (≥500 символов) |
| `task_1/results/encrypted.txt` | Зашифрованный текст |
| `task_1/results/decrypted.txt` | Расшифрованный текст |

### Как запустить:
```bash
cd lab_1/task_1
python main.py
```

### Меню программы:
```
========================================
         МЕНЮ ПРОГРАММЫ
========================================
1. Зашифровать текст
2. Дешифровать текст
0. Выход
========================================
```

### Как проверить:
1. Запустите программу: `python main.py`
2. Выберите пункт `1` (Зашифровать текст)
3. Подтвердите файлы (по умолчанию: `key.txt`, `text.txt`, `results/encrypted.txt`)
4. Выберите пункт `2` (Дешифровать текст)
5. Подтвердите файлы (по умолчанию: `key.txt`, `results/encrypted.txt`, `results/decrypted.txt`)
6. Сравните `results/decrypted.txt` с `text.txt` — должны совпадать

### Формат ключа (key.txt):
Строка пар символ+замена без разделителей:
```
а1б2в3г4д5е6ё7ж8з9и0йык@л#м$н%о^п&р*с(т)у-ф+х=ц\ч/шсщуътыфьоэрю"я<А1Б2В3Г4Д5Е6Ё7Ж8З9И0ЙЫК@Л#М$Н%О^П&Р*С(Т)У-Ф+Х=Ц\Ч/ШСЩУЪТЫФЬОЭРЮ"Я<
```

## Задание 2: Дешифрование по варианту

### Файлы для проверки:
| Файл | Описание |
|------|----------|
| `task_2/cod11.txt` | Зашифрованный текст (вариант 11) |
| `task_2/decrypt.py` | Скрипт с известным ключом |
| `task_2/frequency_analysis.py` | Частотный анализ |
| `task_2/results/decrypted.txt` | Расшифрованный текст |
| `task_2/results/key2.txt` | Найденный ключ |

### Как запустить:
```bash
cd lab_1/task_2
python frequency_analysis.py   # Для подбора ключа
python decrypt.py              # Для расшифровки
```

### Как проверить:
1. Запустите `decrypt.py` — получите расшифрованный текст
2. Откройте `task_2/results/decrypted.txt` — должен быть читаемый текст

---

## Примечания

- Поддерживаются только файлы `.txt`
- Ключ: пары символ+замена без разделителей
- При дешифровании ключ инвертируется автоматически
- Вариант = позиция в списке группы (вариант 11 = `cod11.txt`)
