Выводы по результатам тестирования

Были сгенерированы 3 псевдослучайные последовательности длиной 128 бит
с помощью стандартных ГПСЧ языков C, C++ и Java.

Все последовательности прошли 3 теста NIST:
1. Частотный побитовый тест (Frequency Test)
2. Тест на переключения битов (Runs Test)
3. Тест на самую длинную серию единиц в блоке (Longest Run Test)

Все P-значения превышают порог 0.01, что указывает на хорошие
статистические свойства генераторов.

ВЫВОД: Стандартные генераторы псевдослучайных чисел C, C++ и Java
показывают удовлетворительные статистические характеристики и могут
использоваться для задач, не требующих криптографической стойкости.
