Лабораторная работа №2

Тема: Статистический анализ псевдослучайных последовательностей

Цель работы:

Цель работы — изучить методы генерации псевдослучайных последовательностей и методы проверки их случайности с помощью статистических тестов линейки NIST.

Реализация тестов NIST:

Для проверки случайности последовательностей была разработана программа на языке Python, реализующая три статистических теста.

Программа выполняет:
 1. Частотный побитовый тест
Проверяет, одинаково ли часто встречаются нули и единицы в последовательности.
 2. Тест на одинаковые подряд идущие биты
Проверяет количество переходов между нулями и единицами.
 3. Тест на самую длинную последовательность единиц в блоке
Проверяет максимальную длину последовательности единиц в блоках.

Вывод:

В ходе лабораторной работы были сгенерированы псевдослучайные бинарные последовательности длиной 128 бит с помощью стандартных генераторов случайных чисел языков C++ и Java.

Для проверки случайности полученных последовательностей была написана программа на языке Python, которая реализует три теста из набора NIST: частотный тест, тест на одинаковые подряд идущие биты и тест на самую длинную последовательность единиц в блоке.

После проведения тестирования были получены значения P-value для каждого теста. Так как значения P-value оказались больше порогового значения 0.01, можно считать, что сгенерированные последовательности являются случайными.

Таким образом, цель лабораторной работы была достигнута: были изучены методы генерации псевдослучайных последовательностей и их статистической проверки с помощью тестов NIST.
