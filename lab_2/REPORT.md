### 2. Программная реализация тестов в Python

Программа на Python считывает данные из файлов и вычисляет значение `P-value` для каждого теста.
Согласно методике `NIST`, если $P-value \ge 0.01$, последовательность считается случайной.

#### 2.1 Основные функции тестов:
1. **Частотный тест:** Оценивает близость доли единиц к 0.5.
2. **Тест на серии:** Проверяет частоту смены состояний (из 0 в 1 и наоборот).
3. **Тест на длинную последовательность единиц:** Разбивает 128 бит на 16 блоков по 8 бит и анализирует аномальные скопления единиц.
