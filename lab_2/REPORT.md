# Лабораторная работа №2  
Статистический анализ псевдослучайных последовательностей.

В рамках данной лабораторной работы была реализована проверка
псевдослучайных бинарных последовательностей с использованием
нескольких тестов из набора NIST.

## Что было сделано

Сначала были сгенерированы бинарные последовательности длиной 128 бит
с использованием стандартных генераторов псевдослучайных чисел
языков C++ и Java.

Полученные последовательности были сохранены в файлы:

- sequence1.txt  
- sequence2.txt  
- sequence3.txt  

Далее на языке Python была написана программа, которая выполняет
анализ этих последовательностей.

## Реализованные тесты

В программе реализованы три теста:

- частотный побитовый тест  
- тест на одинаковые подряд идущие биты (тест серий)  
- тест на самую длинную последовательность единиц в блоке  

Каждый тест вычисляет значение P-value, которое используется
для проверки случайности последовательности.

Последовательность считается случайной, если:

P-value ≥ 0.01

## Файлы проекта

Основные файлы проекта:

- Nist_tests.py — основной файл программы, в котором реализованы тесты  
- constants.py — файл с константами программы  
- sequence1.txt, sequence2.txt, sequence3.txt — сгенерированные последовательности  
- results.txt — файл с результатами выполнения тестов  

При запуске программы тесты выполняются для всех последовательностей,
а результаты автоматически записываются в файл results.txt.

## Вывод

В ходе выполнения лабораторной работы была реализована программа
для проверки псевдослучайных последовательностей с помощью тестов
линейки NIST. Проведённый анализ показал, что полученные
последовательности удовлетворяют критерию случайности.
